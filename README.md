# Personal Event Planner App

## Overview
This Android application allows users to create and manage personal events such as meetings, travel plans, and social activities. The app helps users stay organized by storing events locally on the device.

---

## Features

- Add new events with title, category, location, date, and time  
- View all upcoming events sorted by date and time  
- Edit existing events  
- Delete events  
- Data is saved locally using Room Database  

---

## Technologies Used

- Kotlin  
- Android Studio  
- Room Database  
- RecyclerView  
- ViewModel and LiveData  
- Navigation Component  

---

## App Screens

- Event List Screen  
- Add Event Screen  
- Edit Event Option  

---

## How to Run

1. Open the project in Android Studio  
2. Sync Gradle files  
3. Run the app on an emulator or Android device
